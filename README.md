# 📊 Ecommerce Sales Dashboard (Power BI)

##  Project Overview
This project presents an interactive Ecommerce Sales Dashboard created using Power BI to analyze sales performance, profit trends, customer behavior, and payment patterns.

The dashboard provides insights into key business metrics such as total sales, profit, quantity sold, and average order value (AOV), helping users understand business performance through data visualization.


---

## 📈 Key Metrics
The dashboard highlights the following KPIs:

- Total Quantity Sold: 1409  
- Total Profit: 12K  
- Total Sales Amount: 118K  
- Average Order Value (AOV): 33K  

---

## 📊 Dashboard Insights

### Sales by State
Analyzes sales distribution across different states to identify high-performing regions.

### Payment Mode Analysis
Shows how customers prefer to pay using:
- Cash on Delivery (COD)
- UPI
- Credit Card
- Debit Card
- EMI

### Category Analysis
Evaluates product performance across categories:
- Clothing
- Electronics
- Furniture

### Sub-Category Profit Analysis
Shows profit generated by different product sub-categories such as:
- Printers
- Saree
- Chairs
- Trousers
- Electronic Games

### Monthly Profit Trend
Displays profit trends across months to track sales performance over time.

### Customer Contribution
Identifies top customers contributing to total revenue.
